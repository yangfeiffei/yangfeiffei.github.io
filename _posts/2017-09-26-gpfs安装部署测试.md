# 3.升级gpfs

本次升级测试为`3.5.0.3`到`3.5.0.34`。升级过程中应该停止gpfs的服务，升级完成后再全部启动，升级步骤如下：

- 准备好升级包；
- 确认需要升级的节点的mmfs服务关闭；
- 使用rpm安装升级包；
- 重新编译；
- 依次在其他节点进行升级；
- 启动整个gpfs；

# 4.迁移磁盘

可以使用mmadddisk和mmdeldisk两个命令实现为gpfs集群更换新的nsd，在使用mmdeldisk进行删除nsd的时候
会自动进行数据的迁移，但是必须有足够的存储空间。主要步骤如下：

- 增加一个磁盘；
- 创建新的nsd；
- 加入集群中；
- 删除nsd；


```bash
# 这里为c01增加一个新的磁盘
[root@c01 ~]# fdisk -l | grep /dev
```
